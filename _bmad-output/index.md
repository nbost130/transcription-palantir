# Transcription Palantir Documentation

## Project Overview
**Transcription Palantir** is a high-performance, local transcription service using Whisper.cpp. It provides an automated pipeline for ingesting audio files, managing transcription jobs via a priority queue, and exposing results via a REST API and WebSockets.
